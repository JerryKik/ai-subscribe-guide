## 第一步：注册“干净”的尼日利亚 Apple ID

1. 准备一个从未注册过 Apple ID 的新邮箱（Gmail 或 Outlook 都行）。
2. 打开 [Apple ID 官网](https://appleid.apple.com/) 创建账号。
3. 关键信息填写建议：
   - 国家/地区必须选 `Nigeria`。
   - 手机号可用中国地区 `+86`。
   - 地址可用可靠样本模板，确保格式完整。
4. 完成邮箱和手机验证码校验。

如果你遇到 `Cannot create account at this time`，通常是手机号或设备历史记录触发限制。可以去 [Apple Support](https://support.apple.com/) 联系客服，按“Apple ID 问题”走在线聊天或电话流程，说明是新账号创建受限，一般可处理。

## 第二步：只登录 App Store，不登录 iCloud

这个外区账号的定位应该是“支付与订阅工具号”，不是你的主生态账号。最重要的原则只有一条：**只在 App Store 的媒体与购买项目中登录，不要登录 iCloud**。

iOS 18 及以上可参考这条路径：

1. 设置 -> 顶部 Apple 账户 -> iCloud -> 媒体与购买项目 -> 退出当前账号。
2. 打开 App Store，退出原有商店账号。
3. 登录新注册的尼日利亚 Apple ID。
